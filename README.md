# Python AI Assistant 🤖

## About

Python AI Assistant is an intelligent chatbot application built with Python and Artificial Intelligence.

The project demonstrates how Python can be used to create a practical AI-powered assistant capable of understanding user questions and generating responses.

## Features

- AI conversation
- Simple web interface
- Natural language interaction
- Chat history during the session

## Built With

- Python
- OpenAI API
- Streamlit
- GitHub

## Installation

Clone the repository:

```bash
git clone your-repository-link
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

Run:

```bash
streamlit run app.py
```
